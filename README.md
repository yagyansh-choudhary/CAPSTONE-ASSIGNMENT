# Lumina Shop — E-Commerce Web App

A modern, full-featured e-commerce frontend built with React, TypeScript, and Vite. Includes product browsing, cart, wishlist, checkout flow, authentication, and more.

## Tech Stack

- ⚛️ React 18 + TypeScript
- ⚡ Vite
- 🎨 Tailwind CSS
- 🔀 React Router DOM

## Run Locally

**Prerequisites:** Node.js v18+

1. Install dependencies:
   `npm install`
2. Copy the env template and fill in your values:
   `cp .env.example .env.local`
3. Run the development server:
   `npm run dev`

## Features

- 🛍️ Product listing with search & filters
- 🛒 Cart & wishlist with persistent state
- 💳 Multi-step checkout flow
- 👤 Auth (login / register / profile)
- 🌙 Dark / light theme toggle
- 📦 Order confirmation & tracking
